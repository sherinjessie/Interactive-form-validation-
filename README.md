# Interactive-form-validation-
An interactive form validation project that uses HTML, CSS, and JavaScript to check user inputs in real time. It provides instant feedback for each field, making the form easy to use and improving accuracy with a clean and responsive design.
# Run Locally

Prerequisite: Any modern web browser (no Node.js required)

Clone the repository

git clone https://github.com/sherinjessie/interactive-form-validation.git

Navigate to the project directory

cd interactive-form-validation


Run the app
Simply open the index.html file in your browser.
